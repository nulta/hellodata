# Hello, Data!
An open-source project management platform

## Installation
### Requirements
- NodeJS
- MongoDB

### Setup
1. Clone(or download) the source
1. `npm install`
1. `npm run build`
1. Configure the .env file (see [Configure](#configure))
1. `npm start`

### Configure
1. Copy the `.env.example` file
1. Rename the copied file to `.env`
1. Open up the `.env` file, and configure the variables
